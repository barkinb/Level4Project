# Readme

## Build instructions

User manual can be found in manual.md .
### Requirements


* Python 3.10
* Packages: listed in `requirements.txt` 
* Tested on macOS


### Build steps

#### From the terminal/shell:
1) pip install -r requirements.txt
2) cd src/app
3) python3 main.py
